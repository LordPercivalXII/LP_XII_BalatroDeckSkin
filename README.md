# LordPercivalXII's Balatro Deck Skin Mod

This is just a small mod that adds a custom deck face theme, based on characters that I've designed over the years.

## Art Style

Do note that whatever is shown here may differ to what appears ingame, mainly due to scaling and colours shown in game.
